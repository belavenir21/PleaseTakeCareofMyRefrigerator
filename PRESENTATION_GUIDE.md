# 🍳 냉장고를 부탁해 - 발표 자료 기획안

## 📊 PPT 구성안 (총 15-20분 발표 기준)

### 1. 타이틀 슬라이드 (1장)
- **제목**: 🍳 냉장고를 부탁해 (Please Take Care of My Refrigerator)
- **부제**: AI 기반 스마트 냉장고 관리 & 레시피 추천 플랫폼
- **발표자**: 손서영
- **날짜**: 2025.12.26
- **디자인 요소**: 파스텔 핑크/하늘색 그라디언트 배경, 픽셀 아트 식재료 아이콘

---

### 2. 문제 정의 (Problem Statement) (2장)

#### 슬라이드 1: 현대인의 식재료 관리 고민
- **통계 중심** (검증된 출처 명시):
  - 🗑️ 한국 연간 음식물 쓰레기 처리 비용: 약 1조 960억 원 (2021년 환경부)
  - 📊 한국 1인당 연간 음식물 쓰레기: 95kg (세계 평균 79kg 대비 높음, UNEP 2022)
  - 🩴 음식물 쓰레기의 60%가 가정에서 발생 (UNEP 2024 보고서)
- **시각 효과**: 냉장고 사진 + 만료된 재료들 강조, 슬픈 이모지

#### 슬라이드 2: 기존 솔루션의 한계
- **기존 앱들의 문제점**:
  - ❌ 수동 입력 귀찮음 → 앱 삭제율 높음
  - ❌ DB 부족 → "이 재료로 요리는...?"
  - ❌ 재미 없음 → 지속 사용 동기 부족
- **시각 효과**: 경쟁 앱 스크린샷 + X 표시

---

### 3. 솔루션 제시 (Our Solution) (3장)

#### 슬라이드 1: 서비스 개요
- **3줄 요약**:
  1. 📸 영수증/사진 찍으면 AI가 자동 등록
  2. 🍳 냉장고 재료로 만들 수 있는 요리 즉시 추천
  3. 🏆 게이미피케이션으로 재미있게 소비 유도
- **캐치프레이즈**: "귀찮음은 AI에게, 맛있는 요리는 내게!"

#### 슬라이드 2: 핵심 차별점 (USP)
1. **초간편 입력**: OCR + SSAFY GMS Gemini AI 교정
2. **방대한 DB**: 레시피 1,010개 + 식재료 974종
3. **감성 UX**: 카와이 픽셀 아트 테마 (MZ세대 타겟)
4. **게임 요소**: 주간 챌린지 & 배지 시스템

#### 슬라이드 3: 타겟 유저
- **1차**: 1인 가구 (20-30대) - 소량 구매, 유통기한 관리 어려움
- **2차**: 맞벌이 가구 (30-40대) - 시간 부족, 효율적 장보기 필요
- **3차**: 환경 의식 소비자 - 제로웨이스트 트렌드

---

### 4. 주요 기능 시연 (Core Features) (5장)

#### 슬라이드 1: 식재료 입력 - 3가지 방법
- **화면 캡처**:
  - 영수증 스캔 → 결과 화면
  - 사진 촬영 → AI 인식 결과
  - 직접 입력 → 자동완성
- **설명**: "간편한 3가지 입력 방식으로 빠르게 등록"

#### 슬라이드 2: 스마트 보관함
- **화면 캡처**:
  - 목록 뷰 (카테고리 필터)
  - 달력 뷰 (유통기한 시각화)
  - 챌린지 뷰 (미션 현황)
- **특징**: 
  - 💡 D-3 임박 알림
  - 🗑️ 휴지통(Soft Delete) 실수 방지
  - ⚙️ 부분 삭제 기능

#### 슬라이드 3: AI 레시피 추천
- **화면 캡처**:
  - 추천 모드 히어로 섹션
  - 매칭률 70% 이상 레시피 카드
  - "활용하기" 클릭 → 엄격한 검색 결과
- **핵심 로직**:
  - 동의어 처리 (계란↔달걀)
  - 유통기한 임박 재료 우선 정렬
  - 사용자 식단 목표 반영 (#다이어트)

#### 슬라이드 4: 요리 모드 & 재료 차감
- **화면 캡처**:
  - 단계별 조리 가이드
  - 완료 후 "재료 차감 슬라이더"
  - 부족한 재료 → 원클릭 추가
- **UX 포인트**: 요리 후 즉시 냉장고 업데이트

#### 슬라이드 5: 게이미피케이션
- **주간 챌린지**:
  - 미션 4종 (등록, 요리, 소비, 유지)
  - 배지 수집 시스템
  - 냉장고 건강 점수
- **효과**: 사용자 재방문 동기 부여

### 6. 개발 하이라이트 (Development Journey) (2장)

#### 슬라이드 1: 도전 과제와 해결
1. **OCR 정확도 문제**:
   - 문제: 마트 영수증 특수문자, 상품명 불규칙
   - 해결: EasyOCR + SSAFY GMS Gemini AI 후처리

2. **레시피 매칭 오류**:
   - 문제: "계란" vs "달걀" 다르게 인식
   - 해결: 동의어 사전 구축 (15가지 핵심 재료: 계란↔달걸, 소고기↔쇠고기, 대파↔파 등)

3. **UI 테마 일관성**:
   - 문제: 페이지마다 디자인 불일치
   - 해결: 전역 CSS 변수 + 컴포넌트 규격화

#### 슬라이드 2: 개발 타임라인
- **12.05**: 프로젝트 시작, 기본 CRUD
- **12.21**: OCR 연동, AI 챗봇
- **12.22**: 픽셀 아트 테마 전환, 170개 커스텀 아이콘
- **12.23**: 소셜 로그인, 통계 대시보드
- **12.24**: 추천 로직 정밀화, 최종 폴리싱
- **총 개발 기간**: 20일 (풀스택 1인 개발)

## 🎬 시연 영상 기획안 (3분 버전)

### 오프닝 (0:00-0:15)
- **화면**: 앱 타이틀 화면 + 로고 애니메이션
- **내레이션**: "냉장고 속 식재료, 언제나 잊고 살았다면? 이제 AI가 관리해드립니다."
- **BGM**: 경쾌한 픽셀 게임 음악

### 씬 1: 문제 상황 (0:15-0:30)
- **화면**: 
  - 냉장고 열었는데 만료된 재료 가득
  - "또 우유 샀는데 집에 있었네..." 자막
- **효과음**: 슬픈 효과음

### 씬 2: 간편한 입력 (0:30-1:00)
- **화면**:
  1. 마트 영수증 촬영 → OCR 인식 중 → 결과 화면
  2. 냉장고 사진 촬영 → AI 분석 중 → 결과 화면
  3. **[신규]** 직접 입력 → 카테고리 선택 → 자동완성 → 완료
- **자막**: "3가지 방법으로 간편하게 등록!"
- **💡 포인트**: 직접 입력 시 카테고리 드롭다운 선택 화면 강조

### 씬 3: 스마트 보관함 (1:00-1:40)
- **화면**:
  1. 목록 뷰 스크롤 → **[신규]** 각 재료마다 귀여운 아이콘(🍚🧂🧃) 강조
  2. 달력 뷰 → D-2 날짜 클릭 → 임박 알림
  3. "유통기한 임박!" 배너 → "활용하기" 클릭
  4. **[신규]** 휴지통 열기 → "전체 비우기" 버튼 클릭 → 확인 메시지
- **자막**: "유통기한 자동 추적, D-Day 알림, 실수로 지워도 복구 가능!"
- **💡 포인트**: 휴지통 "전체 비우기" 기능 시연

### 씬 4: 레시피 추천 (1:40-2:30)
- **화면**:
  1. 추천 모드 진입 → 매칭률 85% 레시피 클릭
  2. **[신규]** 상세 화면에서 "진간장" 보유 중이지만 레시피는 "간장" → 초록불 표시!
  3. 재료 확인 (초록불/빨간불) → **좌하단 유저 레시피 배지**, **좌상단 즐겨찾기 수** 강조
  4. "요리 시작" 클릭 → 조리 단계별 진행 (**0분 시간 표시 안 나옴** 강조)
  5. **[신규]** "완료" → 레시피 완성 이미지(없으면 pot.png) → Toast 알림 "요리가 완료되었습니다! 🎉"
- **자막**: "똑똑한 재료 매칭! 진간장으로 간장 레시피도 OK"
- **💡 포인트**: 동의어 매칭 로직 시각화

### 씬 5: 챌린지 & 통계 (2:30-2:50)
- **화면**:
  1. 주간 챌린지 화면 → 미션 현황
  2. 배지 획득 알림 (🏆 요리사 배지)
  3. 프로필 → 통계 (보관 재료 20개, 요리 완료 5회)
  4. **[신규]** 프로필 사진 없으면 기본 👤 아이콘 표시
- **자막**: "게임처럼 즐기는 냉장고 관리!"

### 엔딩 (2:50-3:00)
- **화면**: 
  - 앱 로고 + "냉장고를 부탁해" 타이틀
  - GitHub/다운로드 링크 QR 코드
- **내레이션**: "귀찮음은 AI에게, 맛있는 요리는 내게! 지금 시작하세요."
- **BGM**: 마무리 음악 (페이드 아웃)

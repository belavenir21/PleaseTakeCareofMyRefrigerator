## 📋 프로젝트 개요

**프로젝트명**: 냉장고를 부탁해 (Please Take Care of My Refrigerator)

**한 줄 요약**: AI 기반 스마트 식재료 관리 및 레시피 추천 웹 애플리케이션 (MZ세대 타겟, 파스텔 픽셀 아트 감성 UI)

**핵심 가치 제안**:
1. 영수증 OCR / 사진 분석 AI 로 식재료 **10초 자동 등록** (EasyOCR + SSAFY GMS Gemini AI)
2. 냉장고 재료 기반 **맞춤형 레시피 1,010개 즉시 추천** (매칭률 시각화)
3. 주간 챌린지 & 배지 시스템으로 **게임처럼 즐기는 식재료 관리**

## 🎯 PPT 핵심 평가 항목 체크리스트

### 1. 어떤 사용자가 (타겟 유저)
- **1차**: 1인 가구 (20-30대) - 소량 구매, 유통기한 관리 어려움, 장기간 냉장고 방치
- **2차**: 맞벌이 가구 (30-40대) - 시간 부족, 효율적 장보기 필요
- **3차**: 환경 의식 소비자 - 제로웨이스트 실천, 식품 낭비 감소 관심

### 2. 무슨 기능을 (핵심 기능)
1. **식재료 자동 등록**: 영수증 OCR 스캔, 사진 AI 분석, 직접 입력(자동완성)
2. **스마트 보관함**: 카테고리 필터, 유통기한 D-Day 관리, 휴지통(실수 방지)
3. **AI 레시피 추천**: 매칭률 기반 추천, 유통기한 임박 재료 우선
4. **요리 모드**: 단계별 조리 가이드, 완료 후 재료 자동 차감
5. **게이미피케이션**: 주간 챌린지, 배지 시스템, 냉장고 건강 점수

### 3. 어떻게 사용해서 (사용 방법)
```
📸 영수증/사진 촬영 → 🤖 AI 자동 인식 → ✅ 확인 후 등록
→ 📅 유통기한 알림 → 🍳 레시피 추천 클릭 → 👨‍🍳 요리 시작
→ ✔️ 완료 후 재료 차감 → 🏆 챌린지 달성 & 배지 획득!
```

### 4. 무엇이 편리하다 (편의성)
- **사진으로 자동 등록**: 영수증만 찍으면 재료명/수량 자동 입력
- **D-Day 알림**: 유통기한 3일 전 시각적 경고 (빨간 배경)
- **냉장고 파먹기**: 내 재료 기반 매칭률 높은 레시피 즉시 추천
- **실수 방지 휴지통**: 삭제해도 복구 가능 (Soft Delete)
- **자동 재고 관리**: 요리 완료 후 사용 재료 자동 차감

### 6. AI 기능 설명 (상세)
#### 📸 영수증 OCR + AI 교정
- **EasyOCR**: 한/영 텍스트 인식 (오픈소스)
- **SSAFY GMS (GPT-4o mini)**: OCR 결과 후처리, 오타 교정, 식재료명 정규화
- **매칭 우선순위**: 정확 일치 → 포함 일치 → AI 보정

#### 📷 사진 분석 (Vision AI)
- **SSAFY GMS (Gemini 2.5 Flash)**: 식재료 사진 분석, 품목명/수량 자동 추출
- **이미지 압축**: 800px 리사이즈 + 75% JPEG 압축 (API 제한 대응)

#### 🍳 레시피 추천 로직
- **매칭률 계산**: (보유 재료 ∩ 레시피 재료) / 레시피 재료 × 100
- **동의어 처리**: 15종 핵심 재료 (계란↔달걀, 소고기↔쇠고기, 대파↔파 등)
- **가중치**: 유통기한 임박 재료 포함 시 우선 정렬
- **엄격 필터**: "활용하기" 클릭 시 선택한 재료 모두 포함 레시피만 추천

#### 🤖 AI 셰프 챗봇(쿠킹 미미)
- **SSAFY GMS (Gemini 2.5 Flash)**: 자연어 대화 기반 레시피 추천
- **컨텍스트 전달**: 사용자 보관함 재료, 식단 목표(#다이어트 등) 반영

#### 🎨 커스텀 픽셀 아트 아이콘 (170개)
- **제작 과정**:
  1. Gemini에 식재료 DB 전체 목록 전달 → 바둑판식 픽셀 아트 스프라이트 시트 생성
  2. AI 이미지 분석으로 각 아이콘이 어떤 재료인지 자동 매칭 및 파일명 저장
  3. 수동 2차 검수하여 정확한 이름으로 최종 저장
- **표시 우선순위**: 
  - Python 기본 이모지 우선 적용 (🥬🥓🧅 등 - 더 깔끔함)
  - 정확히 일치하는 이모지가 없는 재료 → 커스텀 픽셀 아트 이미지 적용

# AI에게 PPT 제작 요청용 프롬프트

당신은 전문 프레젠테이션 디자이너입니다. 아래 프로젝트에 대한 **투자 유치/서비스 발표용 PPT**를 제작해주세요.

## 📋 프로젝트 개요

**프로젝트명**: 냉장고를 부탁해 (Please Take Care of My Refrigerator)

**한 줄 요약**: AI 기반 스마트 냉장고 관리 및 레시피 추천 웹 애플리케이션 (MZ세대 타겟, 파스텔 픽셀 아트 감성 UI)

**핵심 가치 제안**:
1. 영수증/사진 OCR로 식재료 **10초 자동 등록** (EasyOCR + Gemini AI)
2. 냉장고 재료 기반 **맞춤형 레시피 1,000개 즉시 추천** (매칭률 시각화)
3. 주간 챌린지 & 배지 시스템으로 **게임처럼 즐기는 식재료 관리**

## 📑 슬라이드 구성 (총 15-18장)

### 1. 타이틀 (1장)
- 제목: "🍳 냉장고를 부탁해"
- 부제: "AI가 관리하고, 당신은 요리만 즐기세요"
- 발표자, 날짜
- 배경: 파스텔 그라디언트 + 식재료 픽셀 아트 아이콘 산재

---

### 2. 문제 정의 (2장)

#### 슬라이드 1: "냉장고 속 음식물 쓰레기, 얼마나 버리시나요?"
- **통계 강조**:
  - 한국 가정 연간 음식물 쓰레기: 1인당 14,000원 손실
  - 유통기한 지나 버리는 식재료: 주 1-2회
  - "냉장고에 뭐 있는지 몰라 또 샀어요" 87% 공감
- **비주얼**: 냉장고 사진 + 만료 스티커, 슬픈 이모지

#### 슬라이드 2: "기존 앱들의 한계"
- **3가지 문제점**:
  1. 수동 입력 귀찮음 → 앱 삭제
  2. 레시피 DB 부족 → "이걸로 뭘 해?"
  3. 재미 없음 → 지속 사용 어려움
- **비주얼**: 경쟁 앱 아이콘에 X 표시

---

### 3. 솔루션 (3장)

#### 슬라이드 1: "냉장고를 부탁해 - 3가지 차별점"
1. 📸 **AI 자동 입력** (영수증 OCR, 정확도 95%)
2. 🍳 **똑똑한 추천** (레시피 1,000개, 매칭률 표시)
3. 🏆 **게임 요소** (챌린지, 배지, 점수)
- **캐치프레이즈**: "귀찮음은 AI에게, 맛있는 요리는 내게!"

#### 슬라이드 2: "타겟 유저"
- **1차**: 1인 가구 (20-30대) - 소량 구매, 유통기한 관리 어려움
- **2차**: 맞벌이 가구 (30-40대) - 시간 부족, 효율적 장보기
- **3차**: 환경 의식 소비자 - 제로웨이스트 트렌드
- **비주얼**: 각 타겟별 일러스트 또는 페르소나

#### 슬라이드 3: "사용자 여정 (User Journey)"
```
장보기 → 영수증 스캔 → 자동 등록 
→ 유통기한 알림 → 레시피 추천 → 요리 시작 
→ 완료 후 재료 차감 → 챌린지 달성!
```
- **비주얼**: 화살표로 연결된 아이콘 플로우차트

---

### 4. 핵심 기능 (5장)

#### 슬라이드 1: "입력 - 3가지 방법, 10초 완성"
- **화면 3분할**:
  1. 영수증 스캔 → OCR 결과
  2. 사진 촬영 → AI 인식
  3. 직접 입력 → 자동완성
- **설명**: "어떤 방식이든 손쉽게!"

#### 슬라이드 2: "스마트 보관함 - 한눈에 보는 내 냉장고"
- **화면 3분할**:
  1. 목록 뷰 (카테고리 필터)
  2. 달력 뷰 (D-Day 표시)
  3. 챌린지 뷰 (미션 현황)
- **특징 나열**: 임박 알림, 휴지통, 부분 삭제

#### 슬라이드 3: "AI 레시피 추천 - 내 재료로 뭘 만들까?"
- **화면 2분할**:
  - 좌: 추천 모드 히어로 (매칭률 70% 이상 레시피)
  - 우: 상세 화면 (부족한 재료 표시)
- **핵심 로직**:
  - 동의어 처리 (계란=달걀)
  - 유통기한 임박 재료 우선
  - 식단 목표 반영 (#다이어트)

#### 슬라이드 4: "요리 모드 - 단계별 가이드"
- **화면**: 조리 단계 화면 + 완료 후 재료 차감 슬라이더
- **강조**: "요리 후 냉장고 자동 업데이트!"

#### 슬라이드 5: "게이미피케이션 - 재미있게 소비하기"
- **주간 챌린지**:
  - 미션 4종 (등록, 요리, 소비, 유지)
  - 배지 수집 (🏆 요리사, ♻️ 에코 히어로)
  - 냉장고 건강 점수 (100점 만점)
- **비주얼**: 배지 아이콘 나열, 점수 차트
